# Roxi Cocina

Sitio web estático para el emprendimiento de comida casera **Roxi Cocina**.

El proyecto está desarrollado con:

- HTML
- CSS
- JavaScript vanilla

No usa frameworks, backend ni base de datos. Es compatible con GitHub Pages.
